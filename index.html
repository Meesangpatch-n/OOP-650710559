สิ่งที่เหมือนกันของ Java, C++, Python ในการเขียน OOP

แนวคิดเหมือนกัน

  • มีคลาส (Class), มีวัตถุ (Object), การสืบทอด (Inheritance), การซ่อนข้อมูล (Encapsulation), การพ้องรูป (Polymorphism) เหมือนกัน
  • มีคลาสและวัตถุ: ทุกภาษามีคลาสที่ใช้สร้างวัตถุ
  • สืบทอดได้: เขียนคลาสใหม่แล้วสืบทอดจากคลาสเก่าได้
  • Polymorphism เหมือนกัน: ทุกภาษามีฟีเจอร์ให้เขียนเมธอดชื่อเดียวกัน แต่ทำงานต่างกันในแต่ละคลาสได้

ความแตกต่างกันของ Java, C++, Python ในการเขียน OOP

จัดการหน่วยความจำต่างกัน

  • Java: มีตัวจัดการหน่วยความจำอัตโนมัติ เรียกว่า Garbage Collector พอออบเจ็กต์ไม่ได้ใช้แล้วมันจะลบออกให้เอง
  • C++: ไม่มีอัตโนมัติ ต้องจัดการเอง ต้องคอยใช้ new ตอนสร้างออบเจ็กต์ แล้วก็ใช้ delete เวลาจะลบ
  • Python: จัดการให้เหมือน Java มี Garbage Collection แต่ Python เข้าใจง่ายกว่า

ตัวแปรและชนิดข้อมูล

  • Java: ต้องประกาศตัวแปรพร้อมบอกชนิดข้อมูลก่อนใช้
  • C++: ต้องทำแบบเดียวกับ Java
  • Python: ไม่ต้องประกาศชนิดข้อมูลล่วงหน้า

การสืบทอดหลายคลาส (Multiple Inheritance)

  • Java: ไม่สามารถสืบทอดจากหลายคลาสได้ (ใช้ Interface แทน)
  • C++: สืบทอดจากหลายคลาสได้ตรง ๆ เลย
  • Python: สืบทอดจากหลายคลาสได้ตรง ๆ เลย

การใช้ Interface กับ Abstract Class

  • Java: มีโครงสร้าง Interface และ Abstract Class ชัดเจน
  • C++: ไม่มี Interface โดยตรง แต่ใช้ Pure Virtual Function แทนได้
  • Python: ไม่มี Interface โดยตรง แต่มี Abstract Base Class (ABC) ที่ทำหน้าที่คล้าย ๆ กัน
